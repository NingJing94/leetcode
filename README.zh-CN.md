# leetcode

力扣 playground。

## 特性

* **力扣 playground**，编写解题代码，并编写单元测试加以验证。
* **提交检查**，每次 Git 提交前会运行代码规范检查和单元测试。

## 开始

```shell
# 安装依赖。
$ npm install

# 开启代码变更实时测试。
$ npm start

# 检查代码规范。
$ npm run lint

# 修复代码规范。
$ npm run lint-fix

# 运行单元测试。
$ npm test
```

## 使用

在开始本地开发之前运行 `$ npm start`，这样就可以实时测试你的代码变更。
